# EcoMarker

Proyecto web educativo para el escenario **EcoMarket**: catálogo de productos ecológicos, registro de usuarios y formulario de contacto.

## Estructura
- `index.html`: Página de inicio con hero, features y destacados.
- `productos.html`: Catálogo con búsqueda, filtros y ordenamiento.
- `registro.html`: Registro con validaciones en el cliente.
- `contacto.html`: Formulario de contacto con validaciones.
- `manual_usuario.html`: Manual de usuario (fase 2).
- `styles.css`: Estilos y diseño responsivo.
- `script.js`: Interactividad con JavaScript (filtros, formularios, tema, menú).

## Cómo ejecutar
Abre `index.html` en tu navegador. No requiere backend.

## Cumplimiento de rúbrica
- **HTML semántico**: encabezados, `main`, `section`, `article`, `nav`, `footer`.
- **CSS responsivo**: grid, fluid, mobile menu, dark mode.
- **JS**: validación de formularios, filtros/ordenamiento, toggles accesibles.
- **Funcionalidad**: todas las páginas enlazadas, navegación clara.
- **Presentación**: estructura limpia y texto guía para explicar.

## Personaliza
- Edita los productos en `script.js` (const `PRODUCTS`).
- Reemplaza imágenes en `/assets/`.
- Ajusta colores en `:root` (CSS).
